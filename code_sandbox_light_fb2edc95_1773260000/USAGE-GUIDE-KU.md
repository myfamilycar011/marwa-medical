# 📖 پێڕەوی بەکارهێنان - Usage Guide
## Marwa Medical Website

## 🚀 چۆنیەتی دەستپێکردن (How to Get Started)

### 1️⃣ کردنەوەی وێب سایت (Opening the Website)
- فایلی `index.html` بکەوە لە وێبگەڕێکدا (Chrome, Firefox, Safari)
- یان کلیک بکە لەسەر فایلەکە بۆ کردنەوەی خۆکارانە

### 2️⃣ گەڕان لە کاڵاکان (Browsing Products)
- **سەرەکی**: بەشی سەرەتایی لەگەڵ ناساندنی کۆمپانیا
- **کەتەلۆگ**: هەموو ٨ بەشەکانی کاڵاکان
- **پەیوەندی**: زانیاری پەیوەندیکردن

---

## ✨ تایبەتمەندییە سەرەکییەکان (Main Features)

### 🛍️ زیادکردنی کاڵای نوێ (Adding New Products)

**هەنگاوەکان:**
1. بڕۆ بۆ بەشی پێویست لە کەتەلۆگ
2. کلیک بکە لەسەر دوگمەی **"زیادکردنی کاڵای نوێ"**
3. فۆڕمەکە پڕ بکەرەوە:
   - **ناوی کاڵا**: ناوی کاڵاکە بە کوردی/عەرەبی
   - **نرخ**: نرخی کاڵاکە بە دینار
   - **بەستەری وێنە**: URL ی وێنەی کاڵاکە
   - **وەسف**: وەسفی کورت (ئارەزوومەندانە)
4. کلیک بکە لەسەر **"زیادکردن"**
5. کاڵاکە دەرکەوێت لەو کاتەدا

**تێبینی**: هەموو کاڵاکان لە localStorage پاشەکەوت دەکرێن

---

### 📄 دابەزاندنی کەتەلۆگی PDF (Downloading PDF Catalogs)

**هەنگاوەکان:**
1. بڕۆ بۆ بەشی پێویست
2. کلیک بکە لەسەر دوگمەی **"دابەزاندنی PDF"** 🔴
3. چاوەڕێ بکە چەند چرکەیەک (PDF دروست دەکرێت)
4. فایلی PDF خۆکارانە دادەبەزێت

**ناوەرۆکی PDF:**
- لۆگۆی کۆمپانیا
- ناوی بەش
- هەموو کاڵاکانی بەشەکە لەگەڵ وێنە
- نرخەکان
- زانیاری پەیوەندی

---

### 🔍 گەڕان (Search)

**هەنگاوەکان:**
1. کلیک بکە لەسەر ئایکۆنی گەڕان 🔍 لە سەرەوە
2. نووسینی ناوی کاڵا بنووسە
3. ئەنجامەکان یەکسەر دەرکەون
4. کلیک بکە لەسەر کاڵایەک بۆ پەیوەندی

## 💡 سەرنجەکان (Tips & Tricks)

### ✅ زیادکردنی وێنە (Adding Images)
بۆ وێنەی کاڵاکان، دەتوانیت بەکاربهێنیت:
- **Unsplash**: https://unsplash.com
- **Pexels**: https://pexels.com
- **Imgur**: https://imgur.com
- یان هەر بەستەرێکی تری وێنە

**نموونە**:
```
https://images.unsplash.com/photo-1234567890?w=500
```

### ✅ پاشەکەوتکردنی زانیاری (Data Backup)
کاڵاکان لە localStorage پاشەکەوت دەکرێن. بۆ پارێزگاری:
- PDF ی کەتەلۆگەکان دابەزێنە
- سکرینشۆت لە کاڵاکان بگرە
- زانیاری localStorage کۆپی بکە

### ✅ پاککردنەوەی زانیاری (Clearing Data)
بۆ سڕینەوەی هەموو کاڵاکان:
- بڕۆ بۆ Developer Tools (F12)
- Application → Storage → Local Storage
- سڕینەوەی `marwaProducts`

## 🎯 کێشەی باو (Common Issues)

### ❓ کاڵاکان نامێنێتەوە
**چارەسەر**: دڵنیابە لە Incognito Mode نیت. localStorage لە Private Mode کار ناکات.

### ❓ PDF دانەبەزێت
**چارەسەر**: 
- ڕێگە بدە بە Pop-ups
- Ad blocker ناکارا بکە
- وێبگەڕێکی تر تاقی بکەرەوە

### ❓ وێنە دەرناکەون لە PDF
**چارەسەر**: هەندێک وێنە ڕێگا نادەن بە بەکارهێنان لە PDF. وێنەی تر هەڵبژێرە.

### ❓ گەڕان کار ناکات
**چارەسەر**: دڵنیابە کاڵات زیادکردووە. لانیکەم ٢ پیت بنووسە.
